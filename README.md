# Project Descriptions

This document contains 4 mini software project stories based on real-life scenarios in Salalah.

---

## Project 1 — “Salalah Delivery Express”

A small local delivery company in Salalah Industrial Area is struggling to keep track of their delivery requests.

They want a simple program to store drivers, delivery requests, and customers.

Every delivery has:
- a customer
- a driver
- a status (“Pending”, “On the Way”, “Delivered”)

Rules:
- One driver can have multiple deliveries
- Each delivery has only one driver

---

## Project 2 — “Salalah Book Café”

In Haffa Souq, a cozy café called “Salalah Book Café” offers drinks and allows customers to borrow books while they enjoy their coffee.

They want a system to track:
- menu items (coffee, tea, etc.)
- books
- customers

Rules:
- Each customer can order drinks
- Each customer can borrow max 2 books
- When customer leaves, borrowed books must be marked returned

---

## Project 3 — “Dhofar Car Rental”

A small car rental company near Salalah Airport needs a system to manage car rentals.

The system should contain:
- cars
- customers
- rental records

Each rental record has:
- one customer
- one car
- rental duration (days)
- total cost (car daily rate × number of days)

Rules:
- Cars can be “Available” or “Rented”
- Company wants to check which cars are available
- Company wants to track total generated revenue

---

## Project 4 — “Salalah Sports League”

The Salalah Municipality organizes a yearly Football League between local academies.

They want a system to manage:
- teams
- players
- matches

Each team has:
- a name
- a coach
- a list of players

Each match has:
- two teams
- a score
- a winner

Goal:
- Record match results
- View league standings
